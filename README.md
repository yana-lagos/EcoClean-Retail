🌱 EcoClean API - Backend con Node.js, Express, PostgreSQL y JWT
📌 Descripción de la actividad

EcoClean es un proyecto académico de desarrollo backend construido con Node.js y Express, enfocado en la creación de una API RESTful para la gestión de usuarios y datos de una tienda simulada de productos de limpieza ecológica. El sistema integra arquitectura modular, persistencia en base de datos y autenticación segura mediante JWT.

🎯 Desafío principal

El principal desafío fue desarrollar una API backend completa y escalable, integrando una base de datos relacional con PostgreSQL mediante Sequelize, aplicando arquitectura modular en Node.js y añadiendo mecanismos de autenticación y seguridad con JSON Web Tokens (JWT).

💡 Solución propuesta

Se implementó una API RESTful utilizando Node.js y Express, organizada en capas (routes, controllers y models). Se conectó la aplicación a una base de datos PostgreSQL utilizando Sequelize como ORM, permitiendo operaciones CRUD. Además, se incorporó autenticación mediante JWT para proteger rutas y controlar el acceso a recursos.
